Запрос  доступных для подключения услуг и блокировок

Характеристики API
Название характеристикиЗначение
Проект59234 Public API 3.0
Интерфейс методаhttps://crmapiweb02.msk.mts.ru:8243/PublicAPI/BP05/v3.0.0
ПротоколHTTPS (RESTful)
МетодGET
ФорматJSON
ТипСинхронный
Производительность      см. табличку

Входные параметры (Детальное описание request в прикрепленной документации)

Название параметра ТипОписаниеОбязательныйПример 
channel.idHeaderstringНаименование системы, выданное при регистрации в PublicAPI+[system_name] 
category.nameQuerystringПоле для разделения вызываемых методов+MobileConnectivity 
       
marketSegment.characteristic.nameQuery Маркетинговый тариф абонента+ (**)ProdOfferPurchased 
marketSegment.characteristic.valueQuery Значение маркетингового тарифа абонента-[0699.1.100,0397.2.153911] 
marketSegment.characteristic.name  MSISDN абонента+(**)MSISDN 
marketSegment.characteristic.value  Значение msisdn абонента-[79000000000,79000000001] 
       
productOffering.actionAllowedQuery Тип действия над продуктом+Create, Delete, Update, по умолчанию, None 
productOffering.productSpecification.productSpecificationType.nameQuery Тип продукта: услуга, блокировка-[service, block] 
marketSegment.place.idQuery Код региона абонента+(**)[100, 153911]
fieldsQuery Флаг  
ServiceProductClassification  

Флаг “Класс продукта для МПУ”

- 

** - либо MSISDN, либо пара Регион-Тариф

     

Возвращаемые данные

Название поляТипОписаниеПример
productOffering.productSpecification:  

 

 externalID Код сервисного маркетингового продукта/ Код маркетинговой услуги"PE0005","PE0078","PE0194", "BL0005", "BL0008"
name Название сервисного маркетингового продукта/ Название маркетинговой услуги."Ежемесячная плата за тариф", "Мобильный интернет", "Международный и национальный роуминг", "Добровольная блокировка", "Первоначальная блокировка"
href 

URL описания сервисного маркетингового продукта

" " (только при тип продукта = услуга )
startDateTime 

Дата начала действия сервисного маркетингового продукта/ маркетинговой услуги.

"2018-07-04T22:19:55.000+0000"

ProductSpecificationType.name

 

Тип маркетинговой услуги/  Тип сервисного маркетингового продукта.

"service", "block"

ProductCategory Код листового класса, к которому относится данное МПУ"subscription_fee"

&productOffering.productSpecification.productSpecificationType.name=[Service]
&marketSegment.place.id=[100,153911]
